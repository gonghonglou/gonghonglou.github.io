Say Bye 2020

发表于 2020-12-31 | 分类于 Life

眨眼又到年终

2020，真是感觉过得又快又慢。快的是还没觉察呢这一年就要结束了，感觉今年啥也没干就过去了，仿佛年初在家办公躲避疫情的日子就在昨天；慢的是恍然想起今年发生的什么事好像是很久很久之前发生的一样，不像是在今年，真是神奇。

不过说到底，今年因为疫情都没出去玩，一直呆在家里，感觉好久没出门了。即使回过老家也是因为家里有事情。没有出去玩就像是一直被关在家里。去年因为换工作，所以十一没去成台湾环岛骑行，今年因为疫情又没能去，明天大概率也是没法去了，唉，记得18年去环海南岛的时候鼓励自己的理由是：“如果今年不去，可能往后就去不成，不管是没时间还是没毅力，总会有各种各样的的理由“，果然，台湾之行就是这种情形了。

工作

去年换了工作，又回了上海。本来去年的年终总结该提到这个的，可是去年的时候因为生活不可爱，年终总结都没写。刚到新公司的时候看到公司里的项目工程，和之前工作过的项目工程差别还挺大的，各种代码写的很基础，待改进的地方很多，有技术含量的内容又是少之又少。所以还是造成了很大的焦虑的，当然从头开始组件化，实践组件化过程中各种问题的解决方案，搭建项目基础架构，写出优秀的工具组件，可以做各种优化，这些也是吸引我的地方了。

从创建私有源，拆分第一个私有组件开始，规范组件发版过程；组件开发过程中的分支管理，修改代码后打包即时生效；编写 CocoaPods 插件提高发版效率；组件化分发生命周期的探索；组件化页面路由方案的探索；组件化资源文件管理方案的探索；重写网络请求框架，提高复用性，易用性；重写基础 VC，导航栏控件；重写各种工具类，重构各种业务模块 等等等等

下半年项目开始转 Swift，一直觉得应该要学习 Swift 了，但是总没有一个契机入手。从今年下半年开始项目开始转 Swift，本来以为会花费蛮长时间上手的，结果写了一两天感觉就可以无障碍上手写业务了，当然还是有很多 Swift 语法特性是提前看了很久的文档，还有之后慢慢学习上手的。但终归是开始了 Swift 的工作，上手还是很轻松的。需要的就是后期慢慢深入了。

之后还要努力往底层和性能优化方向多学习：Swift 也需要往更底层学习；卡顿监控，耗时监控，崩溃监控等 APM 工具；优秀的热修方案等

值得欣慰的是今年写了几篇技术博客，创建了 HoloFoundation 写了些开源组件，也让博客和 GitHub 像了点样子。

生活

又回了上海工作，住在家里，和父母一起。不用考虑房租，回家就有饭吃的感觉，在经历了一年杭州的独自居住和现实生活的暴击之后愈发显得珍贵了。

在杭州工作了一年，缴纳了一年社保，通过人才引进落户了杭州。年中的时候开始正式买房的打算，来来回回周末去了好多次杭州，摇了好几次号也不中。经历了西溪公馆的万人摇之后迟迟没有新楼盘开张，好不容易等到有心仪的楼盘“万科-天空之城”开盘，杭州又出新政，判定为热门楼盘的话只能是高层次人才和无房家庭才能参与摇号，无房家庭指的是已婚没房的和三十岁以上单身的。不符合条件的我也就丧失了参与摇号的资格。对摇号买房失去信心之后又跑去杭州看了一圈二手房，发现二手房较于新房又旧又贵又小，不得已还是打算再摇号试试吧，所幸的是接连在拱墅又有好几个位置价格户型都合适楼盘将要开盘，可惜摇了两次都没中。但竟然在十二月的末尾的又一次摇号中摇中了，203号，还挺靠后的，本来感觉希望不大了，最后却是幸运了捡漏抢到了一套。全家都很开心，接着第二天就去付了首付。算下来面积适当，首付没有很多，月供压力也不算大，总之就是挺合适的，终于了了一件大事。

读书/动漫/桌球

好长时间没有静下心了好好读几本书了，今年看了几本技术类书籍也都没有完全看完，零散看了一点小说也没有看完，总之读书这一项最近两年：完败！

倒是动漫看了不少：《鬼灭之刃》、《咒术回战》、刚开始更新的《进击的巨人最终季》，国漫里的《一人之下3》、《全职高手2》、《雾山五行》等等

最开心的就是今年在公司遇到了一位球友了，几乎每个周五下班后都会去打几个小时，有段时间进步还挺明显的，跟着学了挺多走位的意识，虽然打得依旧很菜，多多练习吧

计划

工作方面多往底层基础架构层面深入，找到一些方向深入钻研！继续更新博客，写一些比较有技术深度的文章。继续更新 GitHub，写一些比较有意义的代码。

生活方面不知道环台湾岛骑行什么时候能被安排上，我想，我总会去的，嗯！

少些时间玩手机，多些时间看看书吧。

尽快在 GitHub 上开一个算法仓库，开始刷刷算法，补补一直以来的明显弱点。

嗯，就这样，不要被生活打倒！
